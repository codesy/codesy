How Does It Work?

  1. Add the codesy extension to your browser.
    • Now when you visit an issue on GitHub you'll see the codesy form in the lower right corner.
  2. Decide if you'll fund or fix the issue:
    • How much can fund for someone else to fix it?
    • How much do you need to fix it yourself?
  3. Codesy notifies a developer when their funding request is met.
  4. The developer fixes the bug and submits a claim to the funders for payout.
